Metro Transit Web Application

For this project, I created a web application that shows visitors of Target Field Stadium the upcoming trains leaving from the stadium's stations. To accomplish this, I used JavaScript to connect to the Twin Cities Metro Transit API data and then fed it through the JavaScript framework Vue to display the data in a series of responsive tables created with HTML & CSS. To further the interactivity of the application, users can filter the train data depending on their desired direction and train line.

Hand Coding Responsive Breakpoints

My web design class used this assignment to teach us how responsive breakpoints work. I created a page with HTML and CSS that used hand coded responsive breakpoints to create a blog page that could be easily edited to include new articles and images without additions to the page's CSS. Although this may not be the industry standard way of creating responsive design, I found that understanding the fundamentals has helped guide my web design and allowed me to more easily pickup bootstrap.
